Перед установкой
- Скачать архив Юпи! https://github.com/yupe/yupe/tarball/master
- Распаковать его в каталог доступный веб-серверу (далее по тексту - yupe)
- Создать базу данных для Юпи! - yupe (как пример)
- Проверить и при необходимости создать каталог /yupe/protected/runtime - для хранения временных файлов (запись для веб-сервера)
- Проверить и при необходимости создать каталог /yupe/assets - для хранения опубликованных ресурсов (запись для веб-сервера)
- Проверить файл /yupe/protected/config/db.php - он должен быть доступен для записи веб-серверу и на момент начала установки НЕ должен содержать реальных данных для подключения

Автоматическая установка
- Запустить установщик http://localhost/yupe/index.php/install
- Следовать инструкциям установщика
